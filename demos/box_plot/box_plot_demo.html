1 Introduction

The following R scripts provides some box plot examples of RplotterPkg::create_box_plot(). The dataset being used is from socviz::organdata.

data("organdata")

2 Setting the aes_x and aes_y aesthetic arguments.

Showing outliers labeled with row numbers.

RplotterPkg::create_box_plot(
  df = organdata,
  aes_x = "country",
  aes_y = "donors",
  y_limits = c(0,40),
  y_major_breaks = seq(from = 0, to = 40, by = 5),
  title = "Organ Donation Rate per Million",
  subtitle = "Years 1991 to 2002; outliers labeled with row numbers",
  center_titles = TRUE,
  x_title = "Country",
  y_title = "Donor Rate",
  box_color = "purple",
  box_size = 0.8,
  coord_flip = TRUE,
  rot_y_tic_label = TRUE,
  show_outliers = TRUE,
  ol_color = "red",
  ol_size = 1.5,
  order_by_median = TRUE
)

3 Setting the aes_x and aes_y along with the aes_fill aesthetics arguments.

Not showing outliers which is the default.

df <- organdata %>% 
  filter(!is.na(world))

RplotterPkg::create_box_plot(
  df = df,
  aes_x = "country",
  aes_y = "donors",
  aes_fill = "world",
  y_limits = c(0,40),
  y_major_breaks = seq(from = 0, to = 40, by = 5),
  title = "Organ Donation Rate per Million",
  subtitle = "Years 1991 to 2002; no outliers",
  center_titles = TRUE,
  x_title = "Country",
  y_title = "Donor Rate",
  box_color = "purple",
  box_size = 0.8,
  coord_flip = TRUE,
  rot_y_tic_label = TRUE,
  order_by_median = TRUE
)

4 Labelling outliers with donor rate values

RplotterPkg::create_box_plot(
  df = organdata,
  aes_x = "country",
  aes_y = "donors",
  y_limits = c(0,40),
  y_major_breaks = seq(from = 0, to = 40, by = 5),
  title = "Organ Donation Rate per Million",
  subtitle = "Outliers labelled by donor rate",
  center_titles = TRUE,
  x_title = "Country",
  y_title = "Donor Rate",
  box_color = "purple",
  box_size = 0.8,
  box_fill = "gold",
  coord_flip = TRUE,
  rot_y_tic_label = TRUE,
  label_outlier_var = "donors",
  label_outlier_color = "blue",
  order_by_median = TRUE
